# git 创建分支
## 创建分支
创建一个支名为 js
```
git branch js
```

## 查看分支
```
git branch
```

## 切换分支当
我们进入操作区时，默认的分支为master,我们要进入创建的分支，需要切换
```
git checkout js
```
## 合并分支
当要合并分支时，我要将js分支合并到master中，我们需要先进入master分支。
```
git merge js
```
## 定义标签
```
git tag v1.0 naster "v1.0"
```
创建分支的感觉就是项目组成员分工完成模块，这个模块就相当于分支，再比如，一个人将js写在一个文件夹，将css写在一个文件夹
，然后我们将这两个模块就可以看做两个分支。
